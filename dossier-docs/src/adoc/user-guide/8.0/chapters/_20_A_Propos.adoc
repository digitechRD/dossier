== À propos d'AIRS Dossier

Le menu *À propos de AIRS Dossier* est accessible depuis l’encadré utilisateur :

.Encadré utilisateur
image:20_A_Propos/User_Panel.png[]

En cliquant sur *À propos de AIRS Dossier* le panneau suivant s’affiche :

.Encadré version
image:20_A_Propos/Version1.png[500,500]

On retrouve ici les informations de versions des différents éléments de dossier (application et serveurs). Vous pouvez appuyer sur *Fermer* pour refermer cet affichage.

=== Notes de version

.Accès aux notes de version
image:20_A_Propos/Version2.png[500,500]

Les notes de versions permettent de connaitre les nouveautés apportées à une version spécifique d’{dossier}. Vous pouvez accéder aux *Notes de version* de dossier en cliquant sur le bouton correspondant. Les notes de version s’affichent dans un nouvel onglet de votre navigateur :

.Notes de version
image:20_A_Propos/Note de version.png[800,800]

Vous pouvez cliquer sur le numéro de version pour laquelle vous souhaitez afficher le descriptif des nouveautés correspondantes (8.0.0, 7.2.0 etc.).
