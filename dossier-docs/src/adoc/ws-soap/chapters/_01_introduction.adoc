== Introduction

{dossier} propose depuis sa version *4* des *{webservices}* de type {soap} permettant à d'autres applications ou à des intégrateurs de s'interfacer avec
{dossier}.

La liste des {webservices} disponibles est accessible à l'adresse

[source]
----
http(s)://<hostname>:<port>/<context>/services
----

[IMPORTANT]
====
La plupart des {webservices} sont également accessibles en *REST*. +
Si disponibles, il est conseillé d'utiliser ces *APIs*
====

[NOTE]
====
Les *APIs Rest* sont disponibles à l'adresse `http(s)://<host>:<port>/<context>/rest/services`
====
