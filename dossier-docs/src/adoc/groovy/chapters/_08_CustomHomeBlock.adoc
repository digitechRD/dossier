== CustomHomeBlock

=== Description

Ce script est utilisé pour l'affichage des blocs personnalisables de l'écran d'accueil

_Description_

=== Exemple

Personnalisation du style de l'entête avec un aspect de sous-titre ("h2").

[source, groovy]
----
_model.setHeaderStyleClass("h2")
