=== Administration

[NOTE]
====
Les {webservices} décrits dans ce chapitre permettent d'effectuer des opérations d'administration et de gestion de l'application.

Pour une administration complète de l'application, l'application {adminAirs} reste le point d'entrée privilégié.
====

[IMPORTANT]
====
L'utilisateur connecté/appelant doit posséder le droit *DOSSIER_ADM_AIRS_ADMIN*.
